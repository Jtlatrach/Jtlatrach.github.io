Tutorial de superficies

En este tutorial se muestra paso a paso como crear una superficie usando de herramienta el programa Fusion 360.

Se comienza dibujando con Sketch dos lineas, como se ve en la imagen, se hicieron dos figuras cerradas, una arriba de otra. Tras realizar lo anterior, se hace un loft entre ambas lineas, creando así la superficie que se ve aca. Ahora se realiza un extrude de otra figura cerrada dibujada con Sketch, que atraviesa la superficie anterior. Se realiza un trim, borrando así las aperturas y dejando una superficie con un orificio. Ahora se cierra la superficie, se le hacen fillet en los bordes y se le da un grosor. Tras esto, generamos un corte horizontal para ver los resultados.

Se adjunta video que muestra rápidamente los pasos explicados anteriormente.
